## Exercises

**Exercise 1**
Rewrite the following sentence, replacing the underlined direct object with the correct direct object pronoun.

"María lee *la carta*."

**Exercise 2**
Rewrite the following sentence, replacing the underlined direct object with the correct direct object pronoun.

"Carlos y Ana comen *las manzanas*."

**Exercise 3**
The following sentence has a direct object: "Yo quiero comprar *los zapatos*." Rewrite this sentence using a direct object pronoun in the two possible correct ways.

**Exercise 4**
Your friend asks you, "¿Entiendes la lección de español?" How would you answer "Yes, I understand it" in Spanish, using a direct object pronoun to avoid repeating "la lección de español"?

**Exercise 5**
Translate the following sentence into Spanish, using a direct object pronoun. The verb "to prefer" (*preferir*) is a stem-changing verb (e -> ie).

"We prefer the blue car."

**Exercise 6**
A mother is telling her son what he needs to do. Translate her command into Spanish: "You have to clean your room." Then, rewrite the Spanish sentence using a direct object pronoun in the two possible ways. (Hint: "your room" is *tu cuarto*).

---

## Answer Key

**Answer 1**
**Sentence:** María **la** lee.

**Reasoning:** The direct object is "la carta," which is a feminine, singular noun. The direct object pronoun that corresponds to a third-person, feminine, singular noun is "la." The pronoun is placed directly before the conjugated verb "lee."

**Answer 2**
**Sentence:** Carlos y Ana **las** comen.

**Reasoning:** The direct object is "las manzanas," which is a feminine, plural noun. The direct object pronoun for a third-person, feminine, plural noun is "las." The pronoun is placed before the conjugated verb "comen."

**Answer 3**
**Possible answers:**
1. Yo **los** quiero comprar.
2. Yo quiero comprar**los**.

**Reasoning:** The direct object is "los zapatos," which is a masculine, plural noun, so the correct pronoun is "los." When you have a conjugated verb ("quiero") followed by an infinitive ("comprar"), you have two placement options for the pronoun:
1. Before the conjugated verb.
2. Attached to the end of the infinitive.

**Answer 4**
**Sentence:** Sí, **la** entiendo.

**Reasoning:** The question asks if you understand "la lección." To avoid repeating this phrase, you replace it with a pronoun. "La lección" is feminine and singular, so the correct pronoun is "la." The pronoun goes before the conjugated verb "entiendo" (the "yo" form of "entender").

**Answer 5**
**Sentence:** Nosotros **lo** preferimos.

**Reasoning:**
1.  **Identify the Direct Object:** The direct object is "the blue car" (*el carro azul*), which is masculine and singular. The corresponding pronoun is "lo."
2.  **Conjugate the Verb:** The verb is *preferir* (to prefer). For "we" (*nosotros*), this verb does not have a stem change, so the conjugation is *preferimos*.
3.  **Combine and Place the Pronoun:** Place the pronoun "lo" before the conjugated verb "preferimos" to form the final sentence.

**Answer 6**
**Initial Translation:** Tú tienes que limpiar tu cuarto.

**Rewritten with pronoun (two ways):**
1. Tú **lo** tienes que limpiar.
2. Tú tienes que limpiar**lo**.

**Reasoning:**
1.  **Translate:** "You have to" is an expression using the verb *tener*, specifically *tienes que*. "Clean your room" is *limpiar tu cuarto*.
2.  **Identify Pronoun:** The direct object is "tu cuarto," which is masculine and singular, so the pronoun is "lo."
3.  **Apply Placement Rules:** This sentence has the structure of a conjugated verb (*tienes*) followed by an infinitive (*limpiar*). Therefore, the pronoun "lo" can either be placed before the conjugated verb ("lo tienes que limpiar") or attached to the end of the infinitive ("tienes que limpiarlo").
